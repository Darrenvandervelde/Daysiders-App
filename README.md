## About This Project

Daysiders-App is a digital system that controls different CRM's.  

### Key Features
-  Maintain and categorize Files for easy reading
-  Upload and organize Contact Imformation 
-  Search and filter content quickly
-  Cloud-based storage for accessibility across devices
-  Clean and user-friendly interface

This project aims to provide a **simple yet powerful platform** for anyone who wants to keep their digital content maintained neatly organized.
Its still in its begining development stage.
